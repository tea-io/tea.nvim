# Tea plugin for neovim
To more details about Tea, see [Tea](https://github.com/tea-io/tea)
## Installation
Using [lazy.nvim](https://github.com/folke/lazy.nvim):
```lua
{
    'tea-io/tea.nvim'
}
```
Without package manager:
```lua
vim.opt.runtimepath:prepend(</path/to/tea-nvim>)
```
## Usage
```lua
:TeaConnect
```
